## ğŸ“¥ How to Use

### Importing Configuration Packages

1. **Download the Package**
   - Download the `.rapidstart` or `excel` package file from this repository

2. **Open Business Central**
   - Navigate to: **Configuration Packages** page
   - Or search for "Configuration Packages"

3. **Import the Package**
   - Select **Import Package** or **Import Excel**
   - Choose the downloaded file
   - Review the package contents

4. **Apply the Package**
   - Select the imported package
   - Click **Apply Package**
   - Review and resolve any conflicts
   - Choose whether to apply all data or specific tables
